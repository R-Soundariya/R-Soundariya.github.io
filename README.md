# R. Soundariya — Personal Portfolio

Live at: [r-soundariya.github.io](https://r-soundariya.github.io)

A single-page personal portfolio built with HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies — just open `index.html` in a browser.

## What's inside

- **Home** — intro, skills, and experience summary
- **Products** — project showcase with category filters
- **About** — bio, education, and certifications
- **Contact** — email form and contact details
- **Sound AI** — a built-in chat assistant that answers questions about my work
- Dark mode toggle, fully responsive (mobile + desktop)

## To update content

All content is in `index.html` — search for the section you want to edit and update the text directly. No build step needed.
